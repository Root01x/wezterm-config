# WezTerm Config

Mi configuración personal de [WezTerm](https://wezfurlong.org/wezterm/).

## Características

- **Tema:** Kanagawa Dragon (Gogh)
- **Fuente:** JetBrains Mono 11pt
- **Cursor:** Barra parpadeante
- **Tab bar:** Minimalista con título centrado
- **Opacidad:** 98%

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/Root01x/wezterm-config.git

# Copiar la configuración
cp wezterm-config/.wezterm.lua ~/.wezterm.lua
```

## Requisitos

- [WezTerm](https://wezfurlong.org/wezterm/installation.html)
- [JetBrains Mono](https://www.jetbrains.com/lp/mono/)
